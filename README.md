# HR Analytics Project

## Project Overview
This project analyzes employee data to identify factors affecting employee attrition, job satisfaction, salary distribution, and workforce trends.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Power BI

## Key Objectives
- Analyze employee attrition
- Understand salary patterns
- Explore job satisfaction levels
- Identify workforce trends
- Generate HR business insights

## Key Insights
- Employees working overtime show higher attrition.
- Lower salary employees are more likely to leave.
- Job satisfaction influences employee retention.
- Certain departments experience higher employee turnover.
